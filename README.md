# 캐치데이트

캐치데이트는 모임 인원이 모일 수 있는 날짜를 한 번에 Pick 해주는 서비스입니다.
<br>

모임 날짜를 잡아야 하는데,
멤버마다 가능한 날이 달라서 날짜 정하기가가 어려우셨을텐데요!
모임 멤버가 각각 날짜를 선택하여 모두가 가능한 날짜를 알려드리기 때문에 모임 날짜를 잡기 한결 수월해질 수 있습니다.
이젠 모임 멤버에게 일일이 물어보지 말고 '캐치데이트' 하는건 어떨까요?
<br>
