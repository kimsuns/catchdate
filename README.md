# 캐치데이트

캐치데이트는 모임 인원이 모일 수 있는 날짜를 한 번에 Pick 해주는 서비스입니다.
<br>

모임 날짜를 잡을 때, 모임 인원들마다 가능한 날짜가 달라서 날짜를 정하기 어려우셨을텐데요!
모임 인원이 각각 날짜를 선택하여 모두가 가능한 날짜를 알려드리기 때문에 모임 날짜를 잡기 한결 수월해질 수 있습니다.
<br>
<br>
